# Note Marker - PDF 乐谱音符标注器

基于 Web 的 PDF 乐谱查看与自动音符识别工具。支持低音谱号和高音谱号的音符检测，自动计算音高（固定调/首调），适用于 PC 和 iPad。

## 依赖

```bash
pip3 install flask flask-cors opencv-python numpy
```

## 启动

```bash
cd /Users/tablee/note-marker
python3 server.py
```

服务运行在 `http://0.0.0.0:5555`，支持局域网访问（iPad 等设备可通过 `http://<电脑IP>:5555` 访问）。

PC 本地访问：`http://127.0.0.1:5555`

## 功能总览

| 功能 | 说明 |

## 详细使用说明

### 1. 打开 PDF
- 拖拽 PDF 文件到页面，或点击「打开文件」按钮
- PDF 会以无缝连续滚动模式显示

### 2. 基线校准（推荐首次使用时操作）
1. 点击工具栏「📏 基线校准」按钮，进入校准模式
2. 在乐谱上框选一段**只有五线谱、没有音符**的区域
3. 系统自动检测 5 条谱线的间距，显示「校准成功」
4. 校准数据保存在浏览器 localStorage，下次打开无需重新校准
5. 再次点击「📏 基线校准」退出校准模式

> 校准可以提高谱线检测准确率，尤其在有连音线干扰的情况下效果显著。

### 3. 标记音符区域
1. 点击「🎵 低音谱标记」或「🎶 高音谱标记」进入标记模式
2. 在乐谱上拖拽矩形框选包含音符的区域
3. **框选完成后自动识别音符**，无需额外操作
4. 识别成功的区域边框变为绿色
5. 再次点击模式按钮退出标记模式

> 低音谱区域显示为红色框，高音谱区域显示为蓝色框。音符标记保持对应颜色。

### 4. 音高显示
- 检测到的音符以数字（1-7）显示在音符左侧
- 低音谱音高映射（固定调）：G=5, A=6, B=7, C=1, D=2, E=3, F=4
- 高音谱音高映射（固定调）：E=3, F=4, G=5, A=6, B=7, C=1, D=2

### 5. 固定调 / 首调切换
- 工具栏中间有「固定调 ⇄ 首调」开关
- 切换到首调模式后会出现调性选择器（C~B）
- 设置会保存在 localStorage

### 6. 重新检测单个区域
- 鼠标悬停在绿色区域框上，右下角出现「↻」刷新按钮
- 点击即可重新检测该区域的音符

### 7. 删除标记
- **删除单个区域**：鼠标悬停在区域框上，右上角出现「✕」删除按钮
- **删除单个音符**：点击「✏️ 清除单个标记」进入清除模式，点击音符即可删除
- **清除所有**：点击「🗑️」按钮清除所有标注

### 8. 调试模式
- 点击「🔍 调试」按钮，在新窗口中打开带检测标注的图片
- 可用于排查识别问题

### 9. 导出 PDF
- 点击「📤 导出PDF」按钮
- 导出的 PDF 包含音符标注，不包含框选标记

### 10. iPad 使用
- 在 iPad Safari 中访问 `http://<电脑IP>:5555`
- 支持 Apple Pencil 和触摸操作
- 建议先在 iPad 上做基线校准（iPad 的图片分辨率可能与 PC 不同）

## 音符识别技术说明

- 支持**全音符**（空心）、**二分音符**（空心带符干）、**四分音符**（实心）、**八分音符**（带符尾）
- 使用 OpenCV 轮廓检测 + 形态学处理
- 谱线检测使用水平投影，基线校准可提高准确率
- 音符符头定位使用密度分析 + 最宽行检测
- 支持超出五线范围的音符（上加线/下加线）
